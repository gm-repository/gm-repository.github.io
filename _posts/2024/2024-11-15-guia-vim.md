### **Guia de Sobrevivência do Vim**

O **Vim** é um dos editores de texto mais populares e eficientes, amplamente utilizado por programadores, administradores de sistemas e entusiastas de tecnologia. Apesar de sua curva de aprendizado, o Vim se destaca pela produtividade, minimalismo e capacidade de ser usado em praticamente qualquer ambiente. Este guia é um manual rápido e prático para quem deseja aprender os comandos essenciais e dominar o básico para sobreviver no Vim.

---

### **1. Primeiros Passos no Vim**

#### **1.1 Instalando o Vim**
- **Linux**:
  ```bash
  sudo apt install vim       # Para Debian/Ubuntu
  sudo dnf install vim       # Para Fedora/RHEL
  sudo pacman -S vim         # Para Arch Linux
  ```
- **macOS**:
  - Via **Homebrew**:
    ```bash
    brew install vim
    ```
- **Windows**:
  - Baixe o instalador em: [Vim Downloads](https://www.vim.org/download.php).

#### **1.2 Modos do Vim**
O Vim possui vários modos de operação. Os três principais são:
- **Modo Normal**: Para navegar e executar comandos (`ESC` para entrar neste modo).
- **Modo Inserção**: Para digitar texto (`i` para entrar neste modo).
- **Modo Comando**: Para executar comandos como salvar ou sair (`:` para entrar neste modo).

### **2. Comandos Básicos**

#### **2.1 Navegação Simples**
| **Ação**          | **Comando**       | **Descrição**                              |
|-------------------|------------------|--------------------------------------------|
| Cima              | `k`              | Move o cursor para a linha acima.          |
| Baixo             | `j`              | Move o cursor para a linha abaixo.         |
| Esquerda          | `h`              | Move o cursor para a esquerda.             |
| Direita           | `l`              | Move o cursor para a direita.              |
| Início da linha   | `0`              | Move para o início da linha atual.         |
| Fim da linha      | `$`              | Move para o fim da linha atual.            |
| Próxima palavra   | `w`              | Salta para o início da próxima palavra.    |
| Palavra anterior  | `b`              | Salta para o início da palavra anterior.   |

---

#### **2.2 Inserção e Edição de Texto**
| **Ação**             | **Comando**    | **Descrição**                              |
|----------------------|----------------|--------------------------------------------|
| Inserir antes do cursor | `i`         | Entra no modo de inserção antes do cursor. |
| Inserir no final da linha | `A`       | Entra no modo de inserção no final da linha.|
| Excluir caractere   | `x`            | Exclui o caractere sob o cursor.           |
| Excluir linha inteira | `dd`         | Exclui a linha atual.                      |
| Copiar linha        | `yy`           | Copia a linha atual (yank).                |
| Colar               | `p`            | Cola o conteúdo copiado ou excluído.       |
| Desfazer            | `u`            | Desfaz a última ação.                      |
| Refazer             | `Ctrl+r`       | Refaz a última ação desfeita.              |

---

#### **2.3 Comandos de Arquivo**
| **Ação**             | **Comando**         | **Descrição**                              |
|----------------------|---------------------|--------------------------------------------|
| Salvar arquivo       | `:w`               | Salva o arquivo atual.                     |
| Sair do Vim          | `:q`               | Sai do Vim (se não houver alterações).     |
| Salvar e sair        | `:wq` ou `ZZ`      | Salva o arquivo e sai do Vim.              |
| Forçar saída         | `:q!`              | Sai sem salvar alterações.                 |
| Abrir arquivo        | `:e nome_do_arquivo` | Abre ou edita um arquivo.                  |

### **3. Navegação Avançada**

#### **3.1 Movendo-se pelo Texto**
| **Ação**                 | **Comando**       | **Descrição**                              |
|--------------------------|------------------|--------------------------------------------|
| Próximo parágrafo        | `{` ou `}`       | Move para o início do próximo/parágrafo anterior. |
| Ir para uma linha específica | `:n`         | Move para a linha `n`.                     |
| Pesquisar texto          | `/texto`         | Pesquisa por "texto" no arquivo.           |
| Próxima ocorrência       | `n`              | Vai para a próxima ocorrência da pesquisa. |
| Ocorrência anterior      | `N`              | Volta para a ocorrência anterior.          |

---

### **4. Trabalhando com Múltiplos Arquivos**

#### **4.1 Dividir a Tela**
| **Ação**               | **Comando**       | **Descrição**                              |
|------------------------|------------------|--------------------------------------------|
| Dividir horizontalmente| `:split`         | Divide a janela horizontalmente.           |
| Dividir verticalmente  | `:vsplit`        | Divide a janela verticalmente.             |
| Alternar janelas       | `Ctrl+w` + `w`   | Alterna entre janelas.                     |

#### **4.2 Abrindo Múltiplos Arquivos**
1. Abra o primeiro arquivo normalmente:
   ```bash
   vim arquivo1.txt
   ```
2. Abra outro arquivo em uma nova janela:
   ```bash
   :split arquivo2.txt
   ```

### **5. Personalizando o Vim**

#### **5.1 Editando o Arquivo de Configuração**
O Vim usa um arquivo de configuração chamado `.vimrc`. Aqui estão algumas configurações úteis:

##### **Habilitar Números de Linha**
```vim
set number
```

##### **Habilitar Syntax Highlighting**
```vim
syntax on
```

##### **Habilitar Auto-Indentar**
```vim
set autoindent
set smartindent
```

##### **Configurar Espaços como Tabulações**
```vim
set tabstop=4
set shiftwidth=4
set expandtab
```

### **6. Resolvendo Problemas Comuns**

#### **6.1 O Vim Travou**
- Pressione `ESC` para sair do modo atual.
- Digite `:q!` para sair sem salvar.

#### **6.2 Quer Salvar Mas Não Sair**
- Digite `:w` para salvar o arquivo sem sair do editor.

#### **6.3 Perdeu-se nos Modos**
- Pressione `ESC` repetidamente para voltar ao **Modo Normal**.

### **8. Recursos Úteis**

- **Plugin Manager**: Use o **Vim-Plug** para gerenciar plugins.
  - Instale o Vim-Plug:
    ```bash
    curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
         https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
    ```
  - Exemplo de configuração no `.vimrc`:
    ```vim
    call plug#begin('~/.vim/plugged')
    Plug 'junegunn/fzf.vim'
    Plug 'tpope/vim-fugitive'
    call plug#end()
    ```
- **Aprendizado**: Use o comando interno `:help` para aprender mais sobre o Vim.
